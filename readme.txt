== Description ==

Austrias government is planing to cancel its CERN-membership. This plugin adds a clickable image to support the online petition http://sos.teilchen.at/petition

The online demonstration wiki can be found at http://wiki.die-truppe.com/Online-Demo

== Installation ==

1. Upload `cern-online-demonstration-austria.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
